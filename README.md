## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ahmadruslandia/Flutter-E-Commerce-App.git
   ```
2. Install packages
   ```sh
   flutter pub get
   ```
